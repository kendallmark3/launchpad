Capabilities

CLI commands

  • feature-launchpad <intent-file>
    Runs Stages 1-3 of the pipeline against an intent file, writing artifacts to generated/feature-launchpad/<feature-name>/.
  • feature-launchpad <intent-file> --output-root <dir>
    Same as above, writing artifacts under a different output directory.
  • python3 -m feature_launchpad.dashboard
    Writes a static, read-only status dashboard (env var / API reachability / capabilities) and opens it in a browser.
